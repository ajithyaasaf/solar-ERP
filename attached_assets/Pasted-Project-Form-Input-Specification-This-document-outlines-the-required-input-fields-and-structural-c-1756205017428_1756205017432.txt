Project Form Input Specification 
This document outlines the required input fields and structural changes for the customer-facing form, 
categorized by system type: On-Grid, Off-Grid, Hybrid, Water Heater, and Water Pump. 
⚡    On-Grid System 
1. Field: Floor 
 Input: dropdown box (floor level) – {0 to 4} 
2. Structure 
 Type: Dropdown with two options 
o GP Structure – Selection Box 
 Sub-fields: Drop down box 
 Lower End Height – {0 to 14} 
 Higher End Height – {0 to 14} 
o Mono Rail 
 Sub-options: dropdown box 
 Mini Rail  
 Long Rail 
3. Civil Work 
 Scope: Dropdown 
o Options: 
 Customer Scope 
 Company Scope 
4. Net Meter 
 Scope: Dropdown box 
o Options: 
 Customer Scope 
 Company Scope
